Proyectos

En esta sección, encontrarás una selección de proyectos en los que he trabajado, que muestran mi experiencia en desarrollo de software, diseño de interfaces y soluciones tecnológicas.

Aras App

Sistema de gestión pedagógica, desarrollada para optimizar tareas como registro de usuarios, servicios, blog, control de asistencia, noticias; pagos entre otros. Con tecnologías como Angular, Spring, Java, Swagger, uso de Mappers, JWT, Mockito, Junit, Jaccoco, entre otros.

Restaurant App

Sistema web creado para llevar un mejor control en las ventas, optimización de tareas, registro de clientes, servicio de delivery, entre otros, construido y desplegado de manera local, adaptado a los requerimientos del cliente.

Top Secret App

Landing page y sistema de gestión hotelera que optimiza la administración de reservas, check-ins y check-outs, facilitando una experiencia fluida tanto para el personal como para los huéspedes.

The Viking System

Sistema que facilita la gestión de pedidos, reservas y control de inventario en tiempo real. Esta solución permite a los empleados manejar órdenes de manera eficiente, agilizar el servicio al cliente y optimizar el flujo de trabajo.

T&M TACTICAL

E-commerce especializado en la venta de ropa táctica, diseñado para ofrecer una experiencia de compra segura y eficiente. Permite a los clientes explorar una variedad de productos tácticos, con descripciones detalladas y opciones de filtro.

UPTASK

Aplicación de gestión de Proyectos y tareas, con funcionalidad de estados por tareas y proyectos, recuperación de contraseña por email, autenticación JWT, graficos Chart para la mejor visualización de estado de proyectos.

Luana Store

Landig page tienda de estampados personalizados, como tazas, camisetas, gorras, llaveros, cases; entre otros. Desarrollada con Angular y desplegada en Vercel.
